# Fighter Class (OOP Practice) ⚔️

A simple Python project to practice Object-Oriented Programming (OOP) concepts.  
Two fighters can attack each other, and their health updates accordingly.

---

## Concepts Learned
- Classes & Objects
- Methods
- Object interaction (attack method)
- Printing object status

---

## How to Run
1. Make sure Python is installed.
2. Run the script:
bash
python fighter_class.py


---

## Example
python
f1 = Fighter("Ali", 100, 20)
f2 = Fighter("Reza", 80, 15)

f1.attack(f2)
f2.Show_status()

f2.attack(f1)
f1.Show_status()


---

✅ This project helps understand the basics of OOP in Python, including creating classes, using methods, and interacting between objects.
